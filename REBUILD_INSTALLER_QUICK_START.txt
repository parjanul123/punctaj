TROUBLESHOOTING

Problem: Build fails with "punctaj.spec not found"
Solution: Make sure you're in d:\punctaj directory

Problem: PyInstaller error
Solution: Ensure PyInstaller is installed
  pip install pyinstaller

Problem: ZIP creation fails
Solution: Windows PowerShell might be blocking it
  Run: Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
  Then run build script again

Problem: NSIS installer not created
Solution: NSIS is optional, ZIP installer will work fine
  To enable: Install NSIS from https://nsis.sourceforge.io/
