Problem: "Python not found" or "Python is not recognized"
  Solution:
    1. Install Python 3.8+ from python.org
    2. During installation: CHECK "Add Python to PATH"
    3. Restart computer
    4. Run installer again

Problem: "Administrator privileges required"
  Solution:
    1. Right-click INSTALL_FOR_OTHER_PCs.bat
    2. Select "Run as administrator"
    3. Click "Yes" when prompted

Problem: Installation fails with "cannot create directory"
  Solution:
    1. Make sure you have Administrator privileges
    2. Check that you have at least 500 MB free disk space
    3. Disable antivirus temporarily and try again

Problem: Application won't start after installation
  Solution:
    1. Double-click "Punctaj Manager" shortcut on Desktop
    2. If still doesn't work:
       - Open Command Prompt
       - Type: "C:\Program Files\Punctaj Manager\Punctaj_Manager.bat"
       - Press Enter
       - Check any error messages

Problem: Cloud data doesn't appear
  Solution:
    1. Check internet connection
    2. Wait 30 seconds after launching (first sync takes time)
    3. Check that supabase_config.ini was copied
    4. Restart the application

A: Yes! If multiple people install on different PCs, they share the same
   cloud data and all changes sync automatically.

Q: Is my data secure?
A: Yes, data is encrypted in transit (HTTPS) and stored securely on Supabase.
   Only authorized users can access the data.

Q: What's the system requirement?

   Python 3.8+
   500 MB disk space
   Internet connection
