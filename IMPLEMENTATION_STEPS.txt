CONFIGURATION PATHS (Auto-searched)

Priority order (highest to lowest):
1. Executable directory (where .exe runs from)
2. Script directory (where Python files are)
3. Current working directory
4. C:\Program Files\Punctaj
5. C:\Program Files (x86)\Punctaj
6. %APPDATA%\Punctaj
7. %LOCALAPPDATA%\Punctaj
8. C:\Punctaj
9. D:\Punctaj
10. User home directory

This ensures config is found regardless of installation path!
